Console Calculator: A simple yet extensible command-line calculator that performs basic arithmetic operations. This project demonstrates fundamental C programming concepts such as input validation, control flow, and modular function design.

Features
Supports addition, subtraction, multiplication, and division.
Handles multiple operations in a single expression (e.g., 5 + 3 * 2).
Includes error handling for division by zero and invalid input.
Interactive mode: enter expressions and see results immediately.
Option to read expressions from a file for batch processing.


Tic-Tac-Toe (Two-Player, Console): A classic two-player Tic-Tac-Toe game played entirely in the terminal. Players take turns placing their marks (X and O) on a 3×3 grid, with the program detecting wins, losses, or draws. This project is an excellent exercise in 2D arrays, game logic, and user input handling.

Features
Clean, text-based grid display that updates after each move.
Input validation: prevents moves on already occupied cells and out-of-bounds entries.
Automatic win/tie detection and game-over messages.
Option to restart a new game without exiting.
Simple AI opponent (optional) for single-player mode.


Text-Based Adventure Game: An interactive storytelling game where the player makes choices that shape the narrative. Explore rooms, collect items, and solve puzzles – all within the console. This project introduces concepts like state machines, string handling, and file I/O for loading game content.

Features
Branching storylines with multiple endings.
Rooms (locations) that the player can navigate.
Inventory system: pick up and use items.
Save/load game progress using file I/O.
Modular design: game data (rooms, items, descriptions) can be edited in external text files.
